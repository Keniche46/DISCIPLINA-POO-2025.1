
# Repositório de Práticas em Java – Estrutura Sequencial

Bem-vindo! Este espaço foi criado para reunir uma série de exercícios voltados à prática dos primeiros conceitos de Programação Orientada a Objetos (POO) com foco na linguagem Java. Os conteúdos estão organizados de maneira simples e funcional para facilitar tanto o estudo quanto a prática individual.

## 🎯 Objetivo

O foco principal deste repositório é apresentar resoluções práticas dos exercícios do **Capítulo 03 – Estrutura Sequencial**, retirados da obra _"Fundamentos da Programação de Computadores"_ de **Ana Fernanda Gomes Ascencio** e **Edilene Aparecida Veneruchi**.

## 🔖 Organização dos Arquivos

Os exercícios estão divididos em dois grupos principais:

- **📖 Resoluções** – Questões resolvidas com seus respectivos códigos Java.
- **📝 Propostos** – Questões apenas com enunciados, voltadas para quem deseja praticar.

## 🛠️ Como Utilizar

Para rodar os programas no Eclipse:

1. **Instale o Eclipse IDE**
   - Site oficial: [https://eclipse.org/downloads](https://eclipse.org/downloads)

2. **Crie um novo projeto Java**
   - Vá em `File > New > Java Project` e nomeie o projeto (ex: `POO_EstruturaSequencial`)

3. **Configure o pacote e a classe**
   - Crie o pacote: `br.edu.principal`
   - Crie uma classe dentro do pacote com o nome `Principal`

4. **Cole o código**
   - Acesse o exercício desejado, copie o código-fonte e cole no arquivo `Principal.java`

5. **Execute**
   - Clique com o botão direito no arquivo e selecione `Run As > Java Application`
